# AutoKeyMap

## 功能清单

1. 右Alt键切换中英文输入法，可能需要配置微软拼音键盘布局ID，一般情况下不需要更改
2. 任务栏软件快捷打开，需对任务进行排布
3. 微软拼音下的 VIM IM 自动切换

## 使用说明

需要管理员权限来更好支持按键覆盖，而放在启动目录下的方式无法支持管理员自启动，所以需要用计划任务的方式来开机自启动

运行enableAutoStartUp.bat创建计划任务，disableAutoStartUp.bat删除计划任务

## 遗留问题

在Excel中按RAlt键被excel捕捉而没有触发IM切换, Esc键也没有触发，但是Esc键没有触发是良性的
